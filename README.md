# Explainable KNN Playground (MIT Beaver Works Fall 2025)

Interactive KNN visualizer built during the MIT Beaver Works Fall 2025 Saturday Outreach Program.

Explore how the K-Nearest Neighbors algorithm behaves by adjusting k, distance metric, noise, class weights, and dataset type. Includes guided examples and a quiz mode to teach key ML concepts.

**Live Demo:** https://knn-learning-lab.vercel.app  
**Source Code (this repo):** https://github.com/yanaevteeva2027-code/knn-learning-lab
